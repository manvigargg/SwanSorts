# SwanSorts — AI Waste Classifier 🦢
AI-powered waste classification app built with YOLOv8, FastAPI and React.

## Stack
- YOLOv8 (custom trained model using transfer learning)
- FastAPI backend
- React + Vite frontend
- Supabase (auth + database)

## Run locally
cd backend → pip install -r requirements.txt → uvicorn main:app --reload
cd frontend → npm install → npm run dev
